# Industrial I2C Driver Development & Validation

This repository contains an end-to-end validation framework for a Linux I2C
platform driver developed for Raspberry Pi 4.

## Scope
- Custom Linux I2C platform driver
- Device Tree Overlay integration
- Jenkins-based CI validation
- Functional, stress, and fault testing using open-source tools

## Target Hardware
- Raspberry Pi 4
- I2C LCD 2004A
